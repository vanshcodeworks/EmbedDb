Pipeline walkthrough

  1. WAL append — tensor metadata + floats written to wal.log.
  2. MemTable insert — in-memory, sorted buffer grows until memtableMax.
  3. SST flush — persists to sst_XXXXXX.sst, truncates WAL.
  4. Preview — `TensorStore::summarize` feeds dashboards like this one.
